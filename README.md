Телеграм бот на Python для відстеження зміни ціни збережених товарів

Функціонал: 
- Щоб додати новий товар для відстеження, треба просто надіслати боту посилання на товар з підтримуваних магазинів
- Можна переглянути збережені товари та видалити ті, які більше не цікавлять
- Є функція отримання графіку зміни ціни за останній тиждень з моменту додавання товару
- Оновлення перевіряються тричі на день - в 10, 15 та 20 годин
- Список підтримуваних магазинів є демонстраційним і легко розширюється

Для роботи потрібна база даних на PostgreSQL
Шаблон для файлу .env:

DB_HOST=localhost
DB_PORT=5432
DB_USER=postgres
DB_PASS=password
DB_NAME=your_db_name
BOT_TOKEN=123
LOGGING=True
